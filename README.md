# 💎 Nitro Shine CRM - Management System

System typu **CRM (Customer Relationship Management)** zaprojektowany dla studia auto-detailingu **Nitro Shine**. Projekt łączy logikę biznesową w języku **Python** z relacyjną bazą danych **SQLite**, umożliwiając efektywne zarządzanie procesami usługowymi, bazą klientów oraz finansami studia.

---

##  Kluczowe Funkcjonalności
* **Zarządzanie bazą klientów:** Przechowywanie danych kontaktowych oraz historii pojazdów (marka, model).
* **Katalog usług premium:** Ewidencja usług takich jak korekta lakieru, aplikacja powłok ceramicznych czy detailing wnętrza wraz z cennikiem.
* **System zamówień:** Powiązanie klientów z konkretnymi usługami i terminami realizacji.
* **Raportowanie finansowe:** Automatyczne obliczanie przychodów studia w ujęciu miesięcznym i całościowym.

---

##  Stos technologiczny
* **Język:** Python 3.x
* **Baza danych:** SQLite (Relacyjna baza danych SQL)
* **Biblioteki:** `sqlite3` (standardowa biblioteka Pythona do obsługi SQL)
* **Narzędzia:** Git / GitHub

---

## Struktura Bazy Danych (SQL)
Projekt opiera się na trzech kluczowych tabelach:
1.  `clients` - Dane osobowe i pojazdy.
2.  `services` - Definicje usług i ich ceny.
3.  `orders` - Tabela łącząca (relacja), przechowująca historię zleceń.
